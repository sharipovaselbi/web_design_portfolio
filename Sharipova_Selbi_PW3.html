Ways to open developers console in Mozilla Firefox and Safari

Mozilla Firefox

  • Press F12: Simply press the F12 key on your keyboard to open the developer tools.
  • Right-click and select "Inspect Element": Right-click on any element on the webpage and choose "Inspect Element" from the context menu.
  • Ctrl+Shift+K: Press Ctrl+Shift+K on your keyboard to open the Web Console.

Safari

  • Go to the Safari menu: Click on "Safari" in the menu bar at the top of your screen.
  • Select "Preferences": Choose "Preferences" from the drop-down menu.
  • Go to the Advanced tab: Click on the "Advanced" tab in the Preferences window.
  • Check the box for "Show Develop menu in menu bar": Make sure this option is checked.
  • Access the developer tools: Once you have enabled the Develop menu, you can go to "Develop" in the menu bar and select "Show Web Inspector" to open the developer tools.

Absolute and Relative links

Absolute links use cases

  • When linking to pages on external websites or different domains.
  • When linking to specific pages within your website from external sources or social media platforms.
  • When sharing links externally or across different platforms.
  • When linking to resources like images, stylesheets, or scripts hosted on a different domain.

Relative links use cases

  • When linking to pages within the same website or domain.
  • When you want to maintain flexibility and portability of your website by using paths relative to the current page.
  • When moving or restructuring your website, relative links can adapt automatically without needing to be updated.

I think using relative links is preferable because they are more flexible and easier to maintain, especially when it comes to restructuring or moving content. However, it also depends on the situation.

Image formats

JPG(JPEG)

  • JPG is image format for photographs and complex images on the web.
  • It uses lossy compression, which means it sacrifices some image quality to reduce file size.
  • JPG does not support transparency.
  • JPG supports millions of colors.

GIF

  • GIF is a popular image format for simple animations, logos, icons, and images with limited colors.
  • It supports transparency and animation.
  • GIF images can be animated by combining multiple frames into a single file.

PNG

  • PNG is a versatile image format that supports lossless compression, preserving image quality.
  • PNG is commonly used for logos, illustrations, and images that require high quality and transparency.

WEBP

  • WEBP is a modern image format developed by Google that offers both lossy and lossless compression.
  • It provides smaller file sizes compared to JPG and PNG without compromising image quality.
  • WEBP supports transparency and animation.

SVG

  • - SVG is an XML-based vector image format that is resolution-independent and scalable without losing quality.
  • SVG is ideal for logos, icons, illustrations, and other graphics.
  • SVG files are typically smaller in size compared to raster images like JPG, GIF, and PNG.
  • SVG supports interactivity and animation through CSS and JavaScript.
